## Overview

The repository consists of 2 folders:

### 1. material

Graphic source files

### 2. app

The HTML web application (frontend)

## The App

Install dependencies
`npm install`

Run a local webserver
`npm serve`

ESLint
`npm lint`

Build css and js files
`npm build`
